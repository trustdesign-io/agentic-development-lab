# Project Creation Process (Hybrid v3.0)

Date: 2026-02-26

## Objective

Instantiate new projects under governed Hybrid architecture.

---

## Steps

1. Create new repository
2. Apply branch protection (PR required, approval required)
3. Link to Agentic Development Lab doctrine
4. Configure Local lane (Ollama)
5. Configure Cloud lane (OpenRouter)
6. Validate Issue → PR → Review → Merge workflow
7. Confirm escalation logging works

---

## Success Criteria

- Governance enforced
- Escalation functional
- Logs consistent
